# SCT_DS_3 - Bank Marketing Decision Tree Classification

##  Project Overview

This project builds a Decision Tree Classifier to predict whether a customer will subscribe to a term deposit based on demographic and behavioral information. The Bank Marketing Dataset was used to train and evaluate the machine learning model.

---

##  Objective

To develop a Decision Tree Classification model capable of predicting customer subscription behavior and to evaluate its performance using standard classification metrics.

---

##  Dataset

The project uses the Bank Marketing Dataset from the UCI Machine Learning Repository.

https://archive.ics.uci.edu/dataset/222/bank+marketing

##  Results

- Model Accuracy: **89%**
- Successfully classified customer subscription behavior.
- Generated interpretable decision rules through the decision tree structure.

---

##  Key Insights

- Customer demographic and behavioral attributes can effectively predict subscription decisions.
- The Decision Tree model achieved an accuracy of 89%.
- Certain features have a stronger influence on customer responses.
- Decision Trees provide interpretable decision rules that support business decision-making.
- Machine learning can improve customer targeting and marketing efficiency.

##  Conclusion

A Decision Tree Classifier was successfully developed to predict whether a customer would subscribe to a term deposit. The model achieved strong predictive performance and demonstrated how machine learning can support data-driven marketing strategies.
